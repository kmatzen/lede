Built around the way you actually triage.

📥

Watches everywhere you get pinged

Gmail, GitHub, Slack, Outlook, Google Calendar, Outlook Calendar. Connect any subset; Lede only watches what you give it access to.

🎯

Sorted by what needs you now

Each item is scored 0–10 by Claude — direct asks and outages float to the top, automated noise sinks. The model never sees full message bodies, only headers and snippets.

🌙

Stays out of your way

Quiet hours auto-mute, snooze on demand, dismissed items don't come back. Background refresh keeps the bell honest without flooding you.

🔒

Yours alone

No analytics, no telemetry, no servers we run. Tokens live in your Keychain, summaries cache on your Mac, the only network calls are to the providers you connected and the Claude API key you brought.

Privacy, in one paragraph.

Lede reads only what's needed to triage a message — sender, subject, snippet, label, time. It sends those snippets to Anthropic's Claude API to score and summarize them, then caches the result on your Mac. It never reads message bodies, never uploads anything to a server we run, and never collects analytics. Tokens live in your macOS Keychain and you can disconnect any source instantly.
